# Movie Recommendation System 

## Overview

This project is a Movie Recommendation System built using Flask and Scikit-learn.
It recommends movies based on content similarity using TF-IDF and Cosine Similarity.
The system also includes:
User Authentication (Login/Register)
Dashboard with genre analysis
Similarity-based recommendations
Clean web interface

## Features
Content-based movie recommendation
Movie search functionality
User login and registration (Flask-Login)
Secure password hashing
Dashboard with charts (Chart.js)
Similarity score display
Responsive UI using HTML and CSS
